# SurveyAnalyzer

**SurveyAnalyzer** is a Python-based tool for analyzing survey datasets quickly and visually.  
It provides statistical summaries, charts, and exportable reports for numeric and categorical survey data.  

This project is built using **Python** for the backend and **Flask + JavaScript** for a simple interactive web UI.  
It is designed as a small but high-value project to learn **Python, data analysis, and web development** simultaneously.  

---

## Features

- Load survey CSV datasets (e.g., Nepal QuickStats)
- Analyze numeric and categorical columns:
  - Numeric: mean, median, standard deviation, min/max
  - Categorical: value counts, mode, percentage distribution
- Generate charts for visual representation
- Export results as TXT or CSV reports
- Simple web UI with column selection
- JavaScript interactivity for dynamic results

## Installation

1. Clone the repository:

git clone https://github.com/SulakshanCGhimire/SurveyAnalyzer.git
cd SurveyAnalyzer

2. Create and activate a virtual environment:

python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

3. Install dependencies:

pip install -r requirements.txt

---

## Usage

1. Run the Flask app:
python web_app/app.py

2. Open your browser at http://127.0.0.1:5000/

3. Select a column from the dropdown and click Analyze

4. View the analysis results and charts
